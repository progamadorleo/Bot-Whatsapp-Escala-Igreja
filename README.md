# 🤖 Bot-Whatsapp-Escala-Igreja

Sistema automatizado desenvolvido com a biblioteca **Baileys**, responsável por gerenciar e organizar a escala de membros ao longo do mês de forma inteligente e eficiente.

---

## 📌 Sobre o Projeto

Este projeto tem como objetivo **automatizar completamente o processo de criação de escalas mensais**, utilizando o WhatsApp como principal canal de interação.

Além do bot, o sistema conta com um **frontend completo**, permitindo gerar, visualizar e exportar escalas em PDF com base nas confirmações (votos) coletadas diretamente no WhatsApp.

---

## 🚀 Funcionalidades

- 📅 Geração automática de escalas mensais  
- 🤖 Bot de WhatsApp integrado para coleta de disponibilidade  
- 🗳️ Sistema de votos/respostas dos membros via WhatsApp  
- 🔄 Reorganização automática em caso de mudanças  
- 👥 Gerenciamento de membros (adicionar, editar, remover)  
- 📄 Geração de escala em PDF de forma automática  
- 🌐 Interface web (frontend) para controle e visualização  

---

## 🧠 Como Funciona

1. O bot envia mensagens no WhatsApp solicitando a disponibilidade dos membros  
2. Os membros respondem diretamente no chat  
3. O sistema coleta e processa essas respostas  
4. A escala é gerada automaticamente com base nas regras definidas  
5. Pelo frontend, é possível visualizar e exportar a escala em PDF  

---

## 🛠️ Tecnologias Utilizadas

- **Node.js**
- **Baileys (WhatsApp API não oficial)**
- **JavaScript / TypeScript**
- **React (Frontend)**
- **Geração de PDF (jsPDF ou similar)**
